Minecraft Facts API

Get Random Fact

Example: https://minecraft-facts-api.herokuapp.com/facts/random

Use the /facts/random or /facts/ endpoints to get a random fact about Minecraft.

Response:

{

"id": 13,

"fact": "You can find the end portal by throwing eyes of ender and following the direction they fly in.",

"category": "gameplay"

}

Get Fact by ID

Example: https://minecraft-facts-api.herokuapp.com/facts/1

Use the /facts/:id endpoint to get a particular fact about Minecraft. This will always return the fact with the id that you pass.

Response:

{

"id": 1,

"fact": "Minecraft is a sandbox video game developed by Mojang Studios.",

"category": "about"

}

Get Fact by Category

Example: https://minecraft-facts-api.herokuapp.com/facts/category/about

Use the /facts/category/about or the facts/category/gameplay endpoints to get a random fact from one of the two available categories.

Response:

{

"id": 12,

"fact": "Minecraft's music was composed by Daniel Rosenfeld also known as C418.",

"category": "about"

}
